## Adding a New Project

1. Pick the next available number (currently: **11**)
2. Create `src/content/projects/project-11.md`
3. Create `src/assets/images/projects/project-11/` for images
4. Add a `projectSlug: "project-11"` entry in `index_content.md`
5. Update this table
